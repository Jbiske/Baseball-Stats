
# MLB STATS
Baseball Stats

A small side project for displaying mlb stats and standings using react and a mlb stats api. Using SportsData.io api 



## Deployment

Deployed Link: 



To deploy this project run you need to install axios so the api will work

```bash
  npm i axios
  npm run deploy
 

```
